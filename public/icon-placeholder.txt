PWA Icon Placeholders
PWA ICONS NEEDED FOR PRODUCTION

Create these icon files and place them in /public directory:

Required Icons:
1. pwa-64x64.png       (64×64px)   - Small icon
2. pwa-192x192.png     (192×192px) - Standard PWA icon
3. pwa-512x512.png     (512×512px) - Large PWA icon (also maskable)
4. apple-touch-icon.png (180×180px) - iOS home screen
5. favicon-16x16.png   (16×16px)   - Browser tab
6. favicon-32x32.png   (32×32px)   - Browser tab
7. screenshot-1.png    (1170×2532px) - App store screenshot (optional)

Design Guidelines:
- Background: Dark (#1a1a1a) to match app theme
- Design: Simple Sudoku grid or number "9"
- Format: PNG with transparency where appropriate
- Maskable: 512×512 should work with safe zone (80% content area)

Quick Generation:
npm install -g pwa-asset-generator
pwa-asset-generator source-icon.png public/ --icon-only --background "#1a1a1a"

Or use online tools:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/

The app will work without these, but won't be installable until icons are added.
